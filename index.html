Development log

This is public beta release, some features may be removed and or modified without notice. Here are some points you may have to know before submit a feedback.

Recently added features:

  • Split text by line.
  • Equalize object width / height.
  • Added object alignment control.
  • Cursor now changed to 'text' when you click text insertion button, and 'crosshair' when inserting triangle, circle etc.
  • Change canvas background color by selecting color when no object is selected.
  • Now you can change text properties (font size etc) in a group at once.
  • Publish typography
  • Upload image
  • Autosave, whenever you made change to the canvas, it will be automatically saved. (Local storage browser feature required)
  • Added object list (layer)
  • Duplicate object
  • Set object style in group at once
  • Add more Google fonts

Recently removed features:

  • Dragable canvas, removed
  • Scale object relative to canvas, removed

Bugs, fixes and changes:

  • Facebook connect/account creation is now fixed
  • Color picker sometimes failed to close, now can be manually closed
  • Replace script loader from YepnopeJS to HeadJS, and now use FallbackJS, LOL

Unfinished features:

  • Save as template
  • Load from template
  • Social share

Next possible features:

  • Add more keyboard shortcuts
  • Localization
  • Insert clipart, may be from Openclipart.org
  • Undo/Redo
  • Color pallete
  • Gradient fill
  • Copy object style
  • Add your own fonts
  • Drawing mode

Known bugs:

  • Text object created by duplicate (shift+D) is not editable, you need to reload page to fix this
  • In some condition, object positioning can be frustating
  • You need to reload page manually after successfully connected to Facebook
